## Biblioteca Universal de Prompting 2026

**`biblioteca-universal-prompting-2026.html`** — Un único archivo HTML portable con:

- 669 prompts operativos embebidos (SPEC Framework)
- Buscador con filtros por categoría, riel y nivel
- 21 términos en glosario interactivo
- 3 asistentes entrenados (Custom GPTs + Gemas públicas)
- 6 AI-Native Workflows progresivos + 1 transversal
- Rutina de 15 días (3 semanas × lun-vie × 30 min)
- 3 flujos universales con prompts reales (CV, conversación difícil, decisión compleja)
- Export JSON integrado para Prompster y expansores
- Bilingüe ES/EN · Dark mode · Funciona offline

### Cómo usar

1. Descarga `biblioteca-universal-prompting-2026.html`
2. Abre en cualquier navegador (no necesita servidor)
3. Busca por alias (`/swot`, `/resume`, `/feedback`) o explora por categoría
